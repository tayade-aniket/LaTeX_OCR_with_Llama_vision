# 📌 Overview

LaTeX OCR AI is a modern AI-powered web application that extracts mathematical equations from images and converts them into accurate LaTeX code using Vision Language Models.

The project leverages multimodal AI capabilities to understand complex mathematical notation from:
- Printed equations
- Handwritten formulas
- Scientific expressions
- Mathematical symbols

This application is designed with a clean responsive UI and production-grade architecture suitable for:
- Students
- Researchers
- Data Scientists
- ML Engineers
- Educators

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/LaTeX-OCR-AI.git
cd LaTeX-OCR-AI
```

---

## 2️⃣ Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\\Scripts\\activate
```

### Linux / Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🔑 Setup Environment Variables

Create a `.env` file in the root directory.

```env
GROQ_API_KEY=your_groq_api_key_here
```

---

# ▶️ Run Application

```bash
streamlit run app.py
```
